# Remito PDF Creator

## Requisitos
- Node.js instalado
- Crear una carpeta llamada `remitos_hoy` y colocar allí tus imágenes

## Uso
1. Ejecutar:
   npm install
2. Luego:
   npm start

Genera un archivo PDF con todas las imágenes y abre la página de AskYourPDF.
Las imágenes procesadas se moverán automáticamente a la carpeta `procesados`.
